# Atlas

Atlas 是一個早期開發中的 agent harness 專案。

目標是把公司內部既有的網頁版 LLM（tgenie）包裝成可程式化呼叫的 wrapped API，並參考 opencode 需要的 agent harness，讓原本只能透過網頁操作的 LLM 能被 agent 工作流程使用。

## 目前狀態

- 專案剛初始化。
- 尚未建立可執行程式。
- 尚未定義安裝、建置或測試流程。

## 預期方向

- 建立 tgenie 網頁操作層或 API 包裝層。
- 提供 agent 可呼叫的模型介面。
- 補齊必要的 harness 行為，例如對話、工具呼叫、狀態管理與錯誤處理。
